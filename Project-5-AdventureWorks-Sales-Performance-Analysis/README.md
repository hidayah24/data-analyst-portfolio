# AdventureWorks Sales Performance Analysis

## Project Description
As a Data Analyst for AdventureWorks Corp., I tackled a critical business challenge: **identifying the root causes of a significant profit decline in 2022 despite rising sales volume**, and delivering actionable recommendations for recovery.

### Problem Statement
Profitability dropped sharply from 2021 to 2022. Analyzed multi-dimensional factors including sales trends, customer segments, product performance, returns, and territorial variations to uncover drivers like margin erosion, high-return categories, and underperforming regions.
